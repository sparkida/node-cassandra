Example

Create a new Schema Object

var testSchema = new Cassandra.Schema({
  id: {default: Cassandra.uuid, required: true},
  name: 'text',
  age: 'int',
  username: 'text'
}, {
  primaryKeys: ['username'],
  views: {
    byName: {
      primaryKeys: ['name'],//, 'username'], is implied
      orderBy: {
        name: 'desc',
        username: 'asc'
      }
    }
  }
});

_prepareColumns(columns)

Prepare Schema columns and validate their types this will set the schema's columns array list and set the "default", "required", and "type" settings for the columns, populating schema.columns, Cassandra.Schema.required, Cassandra.Schema.defaults. This will check the list of types against the Cassandra.types.dataTypes
